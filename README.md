STL zu OBJ Konverter
Ein einfaches Werkzeug, das direkt im Browser läuft und STL-Dateien in OBJ-Dateien umwandelt — auch mehrere auf einmal. Speziell für den zahnmedizinischen Workflow mit .dentalProject-Dateien ausgelegt.
→ Hier geht's zum Konverter
(Diesen Link an alle weitergeben, die das Tool nutzen sollen. Keine Installation nötig.)

Was kann das Tool?

Mehrere STL-Dateien gleichzeitig in OBJ umwandeln
Dateien automatisch nach einem festen Schema umbenennen (optional)
Die fertigen OBJ-Dateien direkt im gleichen Ordner speichern oder als ZIP herunterladen
Hell- und Dunkelmodus

Alles passiert auf deinem eigenen Rechner. Es werden keine Dateien hochgeladen, nichts wird gespeichert, niemand sieht deine Daten.

So nutzt du es — Schritt für Schritt
Schritt 1: Dateien hinzufügen
Du hast zwei Möglichkeiten:
Per Drag-and-Drop (empfohlen): Ziehe einen ganzen Ordner mit deinen STL-Dateien in das große gestrichelte Feld oben auf der Seite.
Per Klick: Klick irgendwo auf das gestrichelte Feld. Es öffnet sich ein Datei-Dialog. Wenn du einen ganzen Ordner auswählen willst, klick stattdessen auf den Link „ganzen Ordner wählen" unter dem Feld.

Tipp: Wenn der Ordner eine .dentalProject-Datei enthält, wird sie automatisch erkannt und ihr Name für die Umbenennung übernommen.

Schritt 2: Umbenennung einstellen (optional)
Wenn die Dateien automatisch umbenannt werden sollen:

Schalter „Dateien umbenennen" anschalten
Das Feld Präfix wird automatisch mit dem Namen der .dentalProject-Datei befüllt (kannst du aber ändern)
Das Feld Suffix ist vorbelegt mit -toothmodel_cad (kannst du ebenfalls ändern)

Unter den Feldern siehst du eine Live-Vorschau, wie die fertigen Dateien heißen werden. Und neben jeder Datei in der Liste steht der neue Name direkt mit einem Pfeil daneben.
Beispiel:

STL-Datei heißt 21.stl
Präfix: 2025-12-01_91156-009
Suffix: -toothmodel_cad
Ergebnis: 2025-12-01_91156-009-21-toothmodel_cad.obj

Schritt 3: Speichern
Du hast zwei Buttons unten:
„In ... speichern": Speichert die OBJ-Dateien direkt im gewählten Ordner. Wenn du einen ganzen Ordner reingezogen hast, schreibt das Tool die fertigen Dateien direkt dort hinein — ohne weiteren Dialog. Beim ersten Mal fragt der Browser einmal nach Schreibrechten, danach geht alles automatisch.
„Als ZIP herunterladen": Packt alle OBJ-Dateien in ein ZIP-Archiv und lädt es herunter. Praktisch, wenn du die Dateien jemandem schicken willst.

Hinweise zur Browser-Kompatibilität

Chrome / Edge / Brave: Volle Funktionalität, inklusive direktes Speichern in den Quellordner.
Firefox / Safari: Funktioniert auch, aber das direkte Speichern in den Ordner ist nicht möglich. Stattdessen wird automatisch nur die ZIP-Option angeboten.


Häufige Fragen
Werden meine Dateien irgendwohin hochgeladen?
Nein. Alles passiert ausschließlich auf deinem Rechner im Browser. Die Webseite enthält den kompletten Konverter, es gibt keinen Server, der etwas verarbeitet.
Brauche ich Internet?
Nur einmal, um die Seite zu öffnen. Danach kannst du sogar offline arbeiten — der Browser hat die Seite gespeichert.
Was wenn der Browser nach Schreibrechten fragt?
Beim ersten „In Ordner speichern" fragt Chrome/Edge einmalig, ob die Seite in den Ordner schreiben darf. Auf „Bearbeiten zulassen" klicken — die Erlaubnis gilt nur für diesen einen Ordner und nur für diese Browser-Sitzung.
Was passiert wenn eine Datei mit dem neuen Namen schon existiert?
Sie wird ohne Nachfrage überschrieben. Damit kannst du eine Umwandlung problemlos wiederholen.
Was wenn die Umwandlung einer Datei fehlschlägt?
Die betroffene Datei wird in der Liste rot markiert mit einer Fehlermeldung. Der Rest der Dateien wird trotzdem normal verarbeitet.
Kann ich auch einzelne Dateien umwandeln, ohne den Umbenennen-Schalter?
Ja. Wenn der Schalter aus ist, behalten die Dateien ihren Originalnamen, nur die Endung wechselt von .stl zu .obj.

Wenn etwas nicht funktioniert
Sollte das Drag-and-Drop nicht klappen, ist das fast immer einer dieser Gründe:

Der Browser ist sehr alt → ein aktueller Chrome, Edge oder Firefox löst es
Du hast die Datei lokal von der Festplatte geöffnet statt über den Link → über den Link aufrufen, dann funktioniert es zuverlässig
Strenge Firmen-Sicherheitseinstellungen → nutze den Klick-Weg über „ganzen Ordner wählen", der geht immer
